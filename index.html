O mnie

Jestem absolwentem kierunku Elektroniki na Politechniece Wrocławskiej. Moją pasją jest programowanie i poznawanie nowych technologii. Lubię podejmować nowe wyzwania.

Naukę programowania zaczynałem ucząc się języka C/C++, co sprowadziło się do ruszenia naprzód w kierunku programowania i poznanie różnych technologii. W czasie studiów miałem okazję mieć styczność z Matlabem, programowaniem w Unixie i bazami danych Oracle oraz z różnymi językami programowania Assembler, Java, Python, HTML, CSS, Javascript. Tematem mojej pracy inżynierskiej był „System zarządzania projektami”, który został stworzony jako aplikacja internetowa zrealizowany głownie z Reactem/Reduxem po stronie klienta, a back-end z wykorzystaniem Node.js + MongoDB, co sprawiło, że cała aplikacja została stworzona głownie w oparciu o JavaScript.

Staram się poznać w podstawowym zakresie, a nawet, gdy jest taka potrzeba bardziej się zagłębić z każdą technologią, tak, aby osiągnąć stworzoną aplikacje.

Chciałbym zająć się zawodowo programowaniem, ciągle się rozwijać, pogłębiać swoją wiedzę z programowania i zdobyć doświadczenie w branży IT. Dlatego poszukuję stażu, stabilnej pracy, która pozwoli mi rozwijąć się w tym kierunku.

Umiejętności

Języki programowania:

HTML

jest to hipertekstowy język znaczników na którym buduje się strukturę i zawartość strony.

CSS

Kaskadowy arkusz stylów, którego język wykorzystywany jest do ładniej wizualizacji, prezentacji i rozmieszczenia elementów strony.

JavaScript

Skryptowy język programowania stosowany jako jeden z najpopularniejszych do budowania aplikacji internetowych. Najczęściej do zapewnienia interakcji poprzez reagowanie na zdarzenia, walidacji danych wprowadzanych w formularzach lub tworzenia złożonych efektów wizualnych.

C

Od tego wszystko się zaczeło. Wykorzystwany przez mnie do zabawy z Arduino.

C++

Pomógł mi się dalej rozwinąć i najwięcej używany przeze mnie na studiach m.in. na strukturach danych i złożoności obliczeniowej.

Java

Tutaj głownie wykorzystywany przeze mnie w aplikacji na Androida. W przyszłości mam zamiar stworzyć aplikacje internetowa wykorzystując go do back-endu.

Frameworki:

Node.js

Node.js zaawansowane środowisko zbudowane na silniku JavaScript V8 do tworzenie skalowalnych i wydajnych aplikacji internetowych. Posiada dwustronną komunikację i jest jednowątkowy. Najważniejszą jego cechą jest asynchroniczny mechanizm zdarzeń i nieblokujący się system wejścia/wyjścia.

Express.js

Wykorzystywana z Node.js minimalistyczna i elastyczna struktura. Ułatwia i przyśpiesza implementację serwera Node.js.

Bootstrap

Framework zawierający zestaw stylów CSS, a także rozszerzających jego możliwość skryptów, który umożliwia szybkie budowanie wyglądu strony internetowej, a takze pomaga zachować stronę responsywną.

Biblioteki:

React

Popularna biblioteka umożliwiająca budowanie dynamicznego wyglądu strony. Działa on głownie po stronie przeglądarki, renderując wygląd stron. Moje przygoda z tą biblioteką zaczęła się w tworzeniu aplikacji z Node.js.

Redux

Biblioteka, która ułatwia zarządzaniem stanami aplikacji w Reactcie. Przechowuje stan całej aplikacji w jednym miejscu. Mam z niej skromną wiedzę i jeszcze się jej ucze ...

jQuery

Wykorzystana przy tej stronie do stworzenia efektów wizualnych i działającego formularza kontaktowego.

Bazy danych:

OracleDB

Relacyjna baza danych, jezykiem zapytań jest SQL oraz PL/SQl. Wykorzystywana na jednym semestrze i zubudwałem na niej projekt "ankieta do oceny kursów przez studentów" w opraciu o oracle apex.

MongoDB

Baza danych NoSQL, występuje tutaj możliwość wykonywania zapytań do bazy poprzez JavaScript. W moim przypadku wykorzystywana z biblioteką mongoose.js w node.js.

Dodatkowo:

Ajax

Wykorzystywałem z biblioteką axios i w Node.js do wysyłania i obsługi żądań po stronie użytkownika.

Babel

Babel służy do transpilacji kodu. W przypadku kodu JavaScript z nowszej na starszą wersje przykładowo ES6 na ES5 tak, aby aplikacja była obsługiwana przez większość przeglądarek. Wykorzystywany z Reactem.

Webpack

Webpack potrafi łączyć wiele plików źródłowych w jeden plik wyjściowy. Jeden plik do pobrania z serwera to jedno do niego zapytanie.

Node Package Manager

Posiada bogaty dostęp do bibliotek, które udostępniają wiele gotowych funkcji, co wpływa to na szybkość pisania aplikacji. Za pomocą jednej komendy NPM można zainstalować konkretne rozszerzenie do aplikacji internetowej.

GIT

System kontroli wersji. Wykorzystywany przez mnie z gitkrakenem i za pomocą terminala git bash. Git commit, git push, git pull ...

Matlab

Program, służący do wykonywania obliczeń inżynierskich, oraz do tworzenia symulacji komputerowych. Wykorzystywany na studiach do m.in. przetwarzania obrazów, symulacji układów np. regulatory PID, generowania wykresów
